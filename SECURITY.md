# Security Policy

## 安全报告

如果您发现任何安全漏洞或问题，请通过以下方式报告：

### 如何报告安全问题

请使用 GitHub 的 Private Security Advisory 功能报告安全问题，或者直接联系维护者。

- **重要**：请不要在公开的 GitHub Issue 中报告安全漏洞
- 报告时请包含：
  - 问题的详细描述
  - 复现步骤
  - 可能的影响范围
  - 您的联系方式（可选）

### 安全声明

此项目是一个开源的技能项目目录网站，遵循以下安全原则：

- ✅ 无后端数据库，所有数据静态存储
- ✅ 无用户身份验证系统
- ✅ 无用户输入存储
- ✅ 仅使用 CDN 资源，无自托管依赖
- ✅ 无 Cookie 或本地存储存储敏感信息

### 报告处理流程

1. 收到报告后，我们会在 72 小时内确认
2. 评估问题的严重性和影响
3. 修复问题并发布更新
4. 公开致谢（如果您愿意）
