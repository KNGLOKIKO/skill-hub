# Contributing to Skill Hub

感谢您有兴趣为 Skill Hub 做出贡献！

## How to Contribute

### Reporting Issues

如果您发现任何问题或有改进建议：

1. 检查是否已有相关的 issue
2. 创建新的 issue，包含清晰的描述和复现步骤
3. 提供浏览器类型和版本等相关信息

### Pull Requests

1. Fork 此仓库
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

## Code of Conduct

我们致力于为所有贡献者提供安全和受欢迎的环境。

### Our Standards

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情

### 不允许的行为

- 使用性化的语言或图像
- 恶意评论或人身攻击
- 公开或私下骚扰
- 未经同意发布他人的私人信息
- 其他不专业或不恰当的行为
